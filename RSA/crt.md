# Chinese Remainder Theorem

Convert from $\forall i\in \{1,\cdots,n\} x\equiv a_i\text{ (mod }m_i\text{)}$

to $x\equiv a \text{ (mod lcm}(m_1,\cdots,m_n)\text{)}$

Use the following sagemath code:

```python
crt(a,m)
```
